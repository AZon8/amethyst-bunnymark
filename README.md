# Amethyst BunnyMark

Uses the [Amethyst Game Engine](https://www.amethyst.rs/) to render an increasing number of bunny sprites until 60fps is hit

## Results

| Command             | Bunnies Rendered |
|---------------------|------------------|
| cargo run           | 3,400            |
| cargo run --release | 109,800          |
